MediTrack – Hospital Management System

MediTrack is a Hospital Management System that simplifies hospital operations by managing patients, doctors, appointments, medical records, and billing through a secure and user-friendly platform.

Features

* Patient Management
* Doctor Management
* Appointment Scheduling
* Medical Records Management
* Billing and Payment
* User Authentication
* Admin Dashboard

Technologies Used

* HTML
* CSS
* JavaScript
* Bootstrap
* PHP
* MySQL

Installation

1. Clone the repository.
2. Import the database into MySQL.
3. Configure the database connection.
4. Run the project using XAMPP or WAMP.
